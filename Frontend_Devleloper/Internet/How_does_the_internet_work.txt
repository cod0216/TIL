인터넷 작동 방식 : 개요

높은 수준에서 연결된 디바이스와 켬퓨터 시스템들의 연결로부터 인터넷은 표준화된 프로토콜들의 설정을 함께 사용함으로써 작동한다.
이러한 프로토콜은 어떻게 인터넷이 디바이스들 사이에서 교환되어 지는지와 보장되는지, 전송된 데이터들이 안전하고 신뢰있게 되는지를 정의한다.

인터넷의 핵심은 전 상호 연결된 라우터의 세계적인 네트워크이다. 이러한 라우터는 즉시 응답한다 다른 디바이스들과 시스템들 사이에서. 당신이 인터넷을 통해 데이터를 보낼때, 이 정보들은 작은 패킷들로 부순다. 당신의 디바이스의 라우터로 보내기 위해. 라우터는 패킷을 검사한다 그리고 다름 경로의 라우터에게 보낸다. 이 과정은 연속한다. 패킷이 마지막 지점까지 도달할 때까지.

패킷들이 올바르게 전송되고 전달 받기를 보장하기 위해 인터넷은 인터넷 프로토콜(IP)와 전송 제어 프로토콜(TCP)를 포함한 다양한 프로토콜을 사용한다. IP는 라우팅 패킷이 그들의 올바른 목적지에 도달하는것을 책임지고 있다. 반면에 TCP는 패킷들이 안전하고 올바른 순서에 맞게 전송되는 것을 보장한다.

게다가 이러한 프로토콜은, 넒은 범위의 다른 기술들과 프로토콜은 도메인 시스템(DNS)와 하이퍼텍스트 트렌스펄 포로토콜(HTTP), 그리고 SSL/TLS프로토콜)등을 포함하는 의사소통과 인터넷을 통한 데이터 교환을 가능하게 한다. 계발자로서, 어떻게 이런것들이 다른 기술들이고 프로토콜이 의사소통과 인터넷을 통한 데이터 교환이 함께 작동하는지를 견고하게 이해하는것은 중요하다.

기본 개념과 용어들

인터넷을 이해하기 위해서, 약간의 기본적인 개념들과 용어들에 익숙해 지는것이 중요하다. 여기 몇가지 인식할 수 있는 용어들과 개념들이 있다. 

1. Packet(패킷) : 데이터의 작은 유닛 이것은 인터넷을 통해 전송되어진다.
2. Router(라우터) : 다른 네이워크 사이를 향해하는 데이터 패킷 장치 이다.
3. IP Address(IP 주소) : 네트워크 상에서 다른 디바이스를 위한 배치되는 특별한 식별자이다.특수한 식별자
이다.
4. Domain Name(도메인 네임) : 사람이 읽을 수 있는 이름이다. 이것은 google.com처럼 웹사이트를 식별하기 위해 사용되어진다. 
5. DNS : 도메인 이름 시스템은 도메인 이름을 IP주소로 변환시키는 역할을 한다.
6. HTTP : 하이퍼텍스트 트렌스퍼 프로토콜은 웹 브라우져와 같은 클라이언트와 웹 사이트와 같은 서버 사이에서 데이터를 변환하기 위해 사용되어진다.
7. HTTPS : HTTP의 암호화된 버전이다. 이것은 클라이언트와 서버 사이의 보안적인 의사사통을 제공하기 위해 사용되어 진다.
8. SSL/TLS : 보안 소켓 계층과 전송 계층 보안 프로토콜들이 인터넷을 통해 안전한 의사사통을 제공하기 위해 사용되어진다.

이러한 기본 개념들과 용어들을 이해하는 것은 인터넷 작업과 인터넷 기초 어플리케이션과 서버를 계발함에 있어 필수적이다.

인터넷 프로토콜의 규칙

프로토콜들은 인터넷을 통한 데이터 교환과 케뮤니케이션을 가능하게 하는 역할을 한다. 프로토콜은 어떻게 디바이스와 시스템 사이에서 정보가 교환되어지는지에 대한 표준과 규칙의 집합이다.

인터넷 커뮤니케이션에는 많은 IP, TCP, UDP, DNS 등 다른 프로토콜들이 사용되어 진다. 
IP는 데이터의 라우팅 패킷들이 올바른 목적지에 도착할 수 있도록 책임지고 있다. 반면에 TCP와 DNS는 패킷들이 안전하고 효율적이게 전송되어지게 보장한다. DNS는 도메인 네임을 IP 주소로 변환시켜주며, HTTP는 서버와 클라이언트 사이에서 데이터가 전송되어 지도록 하는 역할을 한다.

표준 프로토콜을 사용하는 이점들 중에 중요한 한가지는 다른 제조사들의 디바이스와 시스템들이 서로 비슷하게 커뮤니케이션을 할수 있게 하는 것이다. 예를들어 회사들끼리 HTTP프로토콜을 고집하는 한 한 회사가 계발한 웹 브라우저가 다른 회사가 계발한 웹 브라우저와 커뮤니케이션 할 수 있다. 

계발자로써, 인터넷 커뮤니케이션에서 사용되는 커뮤니케이션에선 다양한 프로토콜을 이해하는것은 중요하다 그리고 어떻게 프로토콜들이 인터넷을 통한 정보와 데이터가 전송 될 수 있게 작동되어 지는지에 대해 이해하는것이 중요하다.
